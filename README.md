# Free Vending Machine

A tiny console application that simulates a free vending machine. When started the user is shown a list of selections, inputs a number, and the program displays the corresponding vending message.

## Requirements

- .NET 9 SDK
- Visual Studio 2022 (or later) recommended for editing and debugging

## Building and running

Using Visual Studio 2022:
1. Open the solution in Visual Studio 2022.
2. Ensure `VendingMachine1` is the startup project.
3. Start the app with __Debug > Start Debugging__ (F5) or __Debug > Start Without Debugging__ (Ctrl+F5).
